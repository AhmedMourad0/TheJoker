# The Joker

## Project Overview
TheJoker is an app that will tell you one bad joke.

## Key Concepts
- Google Cloud Endpoints, AppEngines, Servlets
- Modularization
- Custom product flavors (free and paid) and build variants
- Custom Gradle tasks
- Google Ads
- AsyncTask, ButterKnife
- Espresso ui tests
